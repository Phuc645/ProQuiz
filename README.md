# ProQuiz

Ứng dụng tạo và thực hiện các bài trắc nghiệm trực tuyến với tính năng tính điểm.

## Tổng quan dự án

- **Thời gian dự kiến**: 3 - 4 tháng
- **Trạng thái**: Đang phát triển
- **Documentation**: [Xem chi tiết](./Docs/SPEC.md)

## Chức năng chính

- Tạo quiz với câu hỏi trắc nghiệm 4 đáp án
- Làm bài quiz của người khác
- Tính điểm
- Quản lý quiz và kết quả

## Công nghệ sử dụng

- **Frontend**: React
- **Backend**: Express.js
- **Database**: PostgreSQL
- **IDE**: WebStorm
- **Cloud**: Google Cloud Platform
