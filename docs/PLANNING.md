# Kế hoạch Phát triển

### Giai đoạn 1: Đặc tả đồ án

- Thời gian: 1 tuần
- Trạng thái: Đã hoàn thành
- Kết quả:
  - Tài liệu đặc tả chi tiết
  - Phác thảo cấu trúc database
  - Định nghĩa các chức năng chính

### Giai đoạn 2: Thiết kế UI/UX

- Thời gian: 2 tuần
- Trạng thái: Đang thực hiện
- Công nghệ: Excalidraw, Figma
- Công việc:
  - Thiết kế wireframe cho các trang chính
  - Thiết kế UI/UX chi tiết
  - Hoàn thiện sơ đồ database

### Giai đoạn 3: Frontend Development

- Thời gian: 4 tuần
- Trạng thái: Chưa bắt đầu
- Công nghệ: React
- Công việc:
  - Xây dựng layout chung
  - Implement các component
  - Testing UI/UX

### Giai đoạn 4: Backend Development

- Thời gian: 4 tuần
- Trạng thái: Chưa bắt đầu
- Công nghệ: Express.js, PostgreSQL
- Công việc:
  - Thiết kế và implement database
  - Xây dựng REST API
  - Integration testing

### Giai đoạn 5: Deployment

- Thời gian: 3 tuần
- Trạng thái: Chưa bắt đầu
- Platform: Google Cloud Platform
- Công việc:
  - Setup môi trường production
  - Configure CI/CD
  - Performance testing
